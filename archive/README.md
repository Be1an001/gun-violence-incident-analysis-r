# Archive Folder

This folder keeps original project materials that are useful for traceability but are not the main public reading path.

## Typical files kept here

- original course reflection
- original model output text
- extra figures not used in the main README
- older or duplicate project materials
- slide-related materials that are kept as original artifacts

## Why this folder exists

I wanted the main repo to feel clean and portfolio-friendly, but I also wanted to keep enough original material to show that this was a real class project with real deliverables.
